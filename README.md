# Keep Marketing Site

## Getting started

### Prerequisites

* Node 20.11.0
* Yarn 1.22.19


### Installation

1. Set correct node version:

```shell
nvm use
```

2. Install all the necessary dependencies:

```shell
yarn install
```

3. Start the application:

```shell
yarn dev
```
